-Применение формул для оперативного расчета показателей.
-Составление сводных отчетов.
-Применение условного форматирования для оперативного анализа.

Описание кейса:
В компании бизнес-процесс Управление персоналом поддерживается в среде MS Excel.
исходный файл - project_task.xlsx

Учет персонала ведется в таблице Учет персонала, где по каждому работающему отражены следующие данные: табельный номер, фамилия, имя, отчество, подразделение, должность, дата рождения, дата приема на работу, дата увольнения, пол, количество иждивенцев, оклад, стаж работы в компании.
По итогам работы за установленный рабочий период сотрудникам могут начисляться баллы за эффективность. Баллы заносятся в таблицу Баллы по итогам рабочего периода и обновляются каждый новый период. В соответствии с баллами установлен процент надбавки к окладу. Сумма надбавки является премией по итогам рабочего периода и рассчитывается автоматически.
Вознаграждение работников включает: оклад, премию по итогам рабочего периода, доплату за стаж. Рассчитывается автоматически в таблице Вознаграждение.
Постоянная, редко обновляемая информация занесена в справочные таблицы. Используются следующие справочники (Лист Справочные данные):
Подразделения – список подразделений компании;
Должностные оклады – установленные для каждой должности оклады. В соответствии с регламентом компании должностные оклады различа-ются для подразделений.
Правила начисления премии – установленные проценты надбавки к окладу в зависимости от количества начисленных баллов.

Для эффективной поддержки бизнес-процесса необходимо:
1. Обеспечить заполнение полей №пп., Отдел, Должность, используя оптимальные технологии ввода;
2. Обеспечить заполнение полей Отдел, Должность, используя данные из справочника отделов и должностей. (Принадлежность сотрудника отделу и должность - произвольные)
3. Обеспечить заполнение поля Оклад на основе данных соответствующего справочника;
4. Обеспечить автоматическое заполнение таблицы Вознаграждение на основе сведений о начисленных баллах и справочных данных о правилах начисления премии. Для расчета надбавки за стаж работы в компании использовать правило: 1000 руб за каждый год работы и дополнительно 3000 для работающих в компании более 5 лет.
Для анализа персонала компании необходимо:
1. Сформировать отчет с информацией о количестве сотрудников в каждом отделе.
2. Сформировать отчет с информацией о величине среднего оклада сотрудников каждого отдела.
3. Сформировать отчет с информацией о всех уволенных сотрудников, занимавших должность секретаря
4. Сформировать отчет со списком всех сотрудников 1971 года рождения.
5. Составить список сотрудников, имеющих оклад меньше среднего.
6. Определить количество иждивенцев в каждом отделе;
7. Сформировать отчет с данными о количестве мужчин и женщин в каждом отделе;
8. Рассчитать средний оклад мужчин и женщин в компании;
9. Найти количество именинников в каждом месяце;
10. В таблице Вознаграждение обеспечить автоматическое выделений цветом записей о работниках, которые получили наибольшую и наименьшую сумму.
